## Running the App with Expo

1. Install Expo CLI (if not already installed)

        npm install -g expo-cli


2. Install Project Dependencies

       cd LemonAID-Mobile
       npm install


3. Start the Server

        npx expo start


4. Scan your QR Code

5. Celebrate
